What is easyPGP?

easyPGP is a cross platform (Mac, Win, Linux) PGP message encrypter / decrypter that aims to make it easier to perform PGP encrypted communication. It is built on Electron and openPGPjs

How it works

First you need to generate a public and private key pair in the generate section. You share your public key with the person you want to communicate with, and they share their public key with you. Never share your private key with anyone. You can both use the other person's public key to encrypt a message. Send this encrypted message to each other and you will be able to decrypt the message with your Private Key and Passphrase (if you set one up). The only way to decrypt a message encrypted with a Public Key is with the matching Private Key. Which makes PGP encrypted communication 100% safe and private, as long as you keep your private key secure.
